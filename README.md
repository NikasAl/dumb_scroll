# Dumb Scroll — лента без смысла

Расширение для **Chromium** (Chrome, Edge, Brave и др.): считает время, проведённое на выбранных сайтах, ведёт **статистику по дням**, показывает **прогресс** относительно вчерашнего дня и подталкивает **уменьшать** это время.

## Возможности

- Учёт времени на заданном списке доменов (соцсети, ленты, видео — как настроите).
- Сравнение **сегодня** с **вчера**: текст, полоса прогресса, мини-график за **7 дней**.
- Настройка списка сайтов и опции «считать только когда окно браузера в фокусе».
- Данные хранятся **локально** в браузере (`chrome.storage.local`).

## Как считается время

Время добавляется, когда одновременно выполняется:

1. Открыта **активная вкладка** с URL, который попадает под правило из списка.
2. Состояние **не idle** (нет активности клавиатуры/мыши в течение интервала, который задаёт Chrome; обычно около минуты — таймер ставится на паузу).
3. Если включена опция фокуса — окно Chrome **в фокусе**.

Отдельно «листать ленту» в коде не детектится: это **приближение** — время на страницах из списка в активной вкладке. Чтобы учитывать только часть сайта (например, ленту LinkedIn), в настройках можно указать путь: `linkedin.com/feed`.

## Установка (режим разработчика)

1. Клонируйте или скачайте эту папку.
2. Откройте `chrome://extensions` (или аналог в вашем браузере).
3. Включите **«Режим разработчика»**.
4. **«Загрузить распакованное расширение»** → выберите каталог `dumb_scroll` (где лежит `manifest.json`).

## Настройки

Правый клик по иконке → **Параметры** / пункт **«Сайты и настройки»** в popup, либо страница опций из списка расширений.

- **Сайты** — по одному домену или `домен/путь` на строку.
- **Считать только когда окно Chrome в фокусе** — если выключить, время может капать, пока браузер открыт в фоне (зависит от ОС и фокуса окна).
- **Сбросить статистику** — обнулить накопленные секунды по дням (список сайтов не трогает).

### Домены по умолчанию

Если настройки ещё не сохранялись, используется встроенный список (его можно полностью заменить в опциях), например:

`vk.com/feed`, `news.mail.ru`, `ok.ru`, `rutube.ru`, `reddit.com`, `threads.net`, `snapchat.com`, `linkedin.com/feed`, `facebook.com`, `instagram.com`, `twitter.com`, `x.com`, `tiktok.com`, `youtube.com`.

## Права (разрешения)

| Разрешение        | Зачем |
|-------------------|--------|
| `storage`         | Сохранение статистики и настроек |
| `alarms`          | Периодический сброс накопленного времени на диск |
| `tabs`            | Понимание активной вкладки и URL |
| `idle`            | Пауза учёта при отсутствии активности |
| `<all_urls>`      | Чтение URL вкладок на обычных сайтах (не для отправки данных наружу) |

Сетевых запросов к сторонним серверам расширение не делает.

## Структура проекта

| Файл | Назначение |
|------|------------|
| `manifest.json` | Manifest V3 |
| `background.js` | Service worker: учёт времени, flush, сообщение `flushNow` для popup |
| `popup.html`, `popup.css`, `popup.js` | Окно по клику на иконку |
| `options.html`, `options.css`, `options.js` | Страница настроек |
| `icons/` | Иконки расширения |

## Версия

Версия указана в `manifest.json` (поле `version`).
